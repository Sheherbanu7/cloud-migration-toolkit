# Cloud Migration Toolkit - Project Plan

## Objective
To simulate DevOps workflow in a multi-cloud environment using CI/CD, Infrastructure as Code, container orchestration, security, and monitoring tools.

## Tech Stack (Dummy App)
- **Language**: Python (preferred) or Node.js
- **Framework**: Flask (for Python) or Express (for Node.js)
- **Databases**: MongoDB or PostgreSQL (local + cloud hosted)
- **Containerization**: Docker
- **CI/CD**: GitHub Actions, Jenkins
- **Cloud Platforms**: AWS (primary), Azure, GCP (inter-op + IaC)
- **IaC**: Terraform, Ansible
- **Kubernetes**: EKS / GKE (minikube or kind for local dev)
- **Monitoring**: Prometheus, Grafana, ELK
- **Security**: AWS IAM, Azure Policies, Vault (secrets mgmt)

## Phase Structure
- Phase 1: Foundation 
- Phase 2: Build Dummy App + Git Integration
- Phase 3: CI/CD Automation
- Phase 4: Cloud Infra Setup (IaC)
- Phase 5: Containerization + Orchestration
- Phase 6: Security and Monitoring
- Phase 7: Documentation and Polishing
